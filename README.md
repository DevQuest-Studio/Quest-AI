# Quest AI

**Quest AI** is an interactive AI chat interface featuring two unique personalities: **Cooper** for creative tasks and image generation, and **Codey** for coding and development assistance. Built with **HTML, CSS, and JavaScript**, it offers a sleek, responsive UI with multiple modes and advanced chat features.

---

## Features

- **AI Personas**
  - **Cooper**: Creative ideas, images, and general chat.
  - **Codey**: Code generation, programming guidance, and technical support.
- **Modes**
  - Chat
  - Code generation
  - Image generation
  - Video script generation
- **Dynamic UI**
  - Glassmorphism panels
  - Animated background blobs
  - Jumping dots loader
  - Scrollable chat container
- **Code Support**
  - Markdown rendering
  - Syntax highlighting with Highlight.js
  - HTML/CSS/JS live preview
- **Custom Language Settings**
  - Select preferred language for AI prompts
- **Chat Management**
  - Rename chats
  - Save user settings
  - Start new conversations
- **Responsive Design**
  - Desktop and mobile-friendly
  - Mobile sidebar toggle
